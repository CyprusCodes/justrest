Create elegant and fast Node.js APIs in seconds with Xest

DocumentationGet Started
Get your API up and running in 3 steps 🚀 

npm i xest -g
xx my-xest-api
cd my-xest-api && xx run

Easy to Use

Xest framework is designed from the ground up to be easily installed and used to build your REST API up and running quickly.

CLI Tool

Modern CLI tool lets you use your CLI for generating tables, seed-data, do your migrations. No need to worry about complex data structures.

Built in authentication

Xest framework comes with built-in authentication, which allows you to save a lot of time.

All required packages in one place

Bind your data with single framework. XEST allows to connect to different data sources like MySQL databases, Rest APIs, Docker, etc.

Easy configuration

Xest has multiple preconfigured built in features to allow you to build your API. You can form tables, edit seed-data according to your needs.

Built with NPM, Express JS, Node.js and Docker

Uses Express JS and Node.js. Xest framework is built on top of Express.js, which is one of the most popular Node.js libraries and it takes a “convention over configuration” approach which means developers are not required to create the build process themselves.
